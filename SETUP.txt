# PS2Claw Setup Guide - Retail PS2

## Two Options

### Option A: FreeDVDBoot (Easier - No modding required!)
Uses a burned DVD to exploit the DVD Player. Works on unmodified PS2s.

### Option B: FreeMCBoot (Memory card hack)
Requires FreeMCBoot installed on a memory card.

---

## Option A: FreeDVDBoot (Recommended)

### What You Need
- PS2 Slim (or some Bravia TVs)
- DVD-R disc (DVD-R preferred, not CD)
- DVD burner
- USB storage device
- Ethernet cable + router

### Step 1: Download FreeDVDBoot
1. Go to: https://github.com/CTurt/FreeDVDBoot
2. Download: **PREBUILT ISOs/All PS2 Slims - English language.iso**
3. Set your PS2 language to English first (boot without disc, press Circle > System Config > Language)

### Step 2: Add PS2Claw to the DVD
1. Download the ZIP from GitHub and extract
2. Navigate to: Filesystems/All PS2 slims (3.10 + 3.11) - English language/
3. Create folder: `USB:/boot/PS2CLAW.ELF` (or copy ELF to that folder in the filesystem)
4. Actually - easier method: Just copy PS2CLAW.ELF to the root of that folder

## Step 2: Set Up Network

### Option A: Wired Ethernet (Recommended)
1. Get a **USB Ethernet Adapter** compatible with PS2:
   - **LAN adaptor**: Sony SCPH-10181 (official)
   - **Generic**: AX88172-based USB to Ethernet adapters work

2. Connect Ethernet cable from PS2 to router

3. Configure network in FreeMCBoot or OpenPS2Loader

### Option B: WiFi (More Complex)
- Use a **WiFi adapter** (more expensive, less compatible)
- Or use a **Powerline adapter** (Ethernet over house wiring)

---

## Step 3: Prepare USB Drive

1. Format USB drive as FAT32

2. Create folder structure:
   ```
   USB:/boot/PS2CLAW.ELF
   ```

3. Copy `PS2CLAW.ELF` to that location

---

## Step 4: Load PS2Claw

### Method A: OpenPS2Loader (Recommended)
1. Launch **OpenPS2Loader** from FreeMCBoot
2. Select your USB device
3. Navigate to `boot/PS2CLAW.ELf`
4. Press X to run

### Method B: SMS Media Player
1. Launch **SMS** from FreeMCBoot
2. Navigate to USB > boot > PS2CLAW.ELF
3. Select "ELF File" to run

1. Type `demo` and press Enter

### Commands:
- `demo` - Toggle demo/live mode
- `help` - Show commands

## Network Troubleshooting

**"Network: OFFLINE"**
- Check Ethernet cable is connected
- Verify router has DHCP enabled (default)
- Try a different USB port on PS2

**"Request failed"**
- Check your internet connection
- The API key is hardcoded - should work
- Try typing `demo` to switch back to offline mode

---

## How It Works

1. **PS2Claw** runs on the MIPS R5900 CPU (300MHz)
2. Uses **libcurl** for HTTPS requests to OpenRouter API
3. Sends your text to **Google Gemini** AI
4. Receives and displays the AI response

---

## Troubleshooting

**Black screen after loading:**
- Try different USB port (front vs back)
- Ensure ELF is in correct folder
- Try OpenPS2Loader instead of SMS

**Network not working:**
- Use official Sony LAN adapter for best compatibility
- Check cable is good
- Router must have DHCP (most do)

**Slow responses:**
- PS2 network is 10/100Mbps max
- AI API takes time to respond
- Be patient - it's 1999 hardware! 😅
